# OutVoice - Tasks

## 1. Sign up
  - Create a signup authentication API.
    - Fields:
      - Full name (required)
      - Email address (required if mobile number is not provided)
      - Mobile number (if email is not provided)
      - Password
      - Organization name
      - Tax & compliance (can be same as user's name)
      - PAN Number, GST Number, TAN Number (optional)
